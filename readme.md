# Basic JSX Interpolation

1. String
2. variable
3. Function call

```react
      <div className="App">
          {/* 1. string */}
          {'this is message'}
          {/* 2. variable */}
          {count}
          {/*3. function call*/}
          {getName()}
      </div>
```
